# 🧠 Análise de Dados: Dashboard Interativo de Vendas de Games no Power BI

Este projeto faz parte do Módulo Power BI do curso de Cientista de Dados da EBAC e tem como objetivo a construção de um dashboard interativo no Power BI para análise de vendas no mercado de games.

A base de dados utilizada contém informações sobre vendas de jogos ao longo dos anos, permitindo analisar o desempenho por gênero, categoria, país e publisher, além da evolução temporal das vendas.

## 🧩 Etapas Desenvolvidas

**1. Importação e tratamento dos dados:** Carregamento da base no Power BI e ajustes necessários para análise.

**2. Modelagem dos dados:** Organização das métricas e dimensões para construção das visualizações.

**3. Construção do dashboard:** Desenvolvimento de gráficos interativos e painéis visuais.

**4. Criação de filtros dinâmicos:** Implementação de filtros por produto, categoria e intervalo de anos.

## 📊 Visualizações e Análises

- Vendas por país: \
Análise da distribuição geográfica das vendas ao longo do tempo.

- Vendas por categoria: \
Comparação entre diferentes plataformas (ex.: Wii, DS, X360).

- Vendas por gênero: \
Identificação dos gêneros com maior volume de vendas.

- Vendas por produtos (publisher): \
Avaliação das empresas com maior participação no mercado.

## 📈 Resultados e Insights

- Destaque para gêneros como Shooter e Platform, com maiores volumes de vendas.

- Forte concentração de vendas em América do Norte e Europa.

- Predominância de grandes publishers como Nintendo, Activision e Sony.

- Variação significativa nas vendas ao longo dos anos, com picos em períodos específicos.

## ✅ Conclusão

O dashboard desenvolvido no Power BI permite uma análise clara, dinâmica e interativa do mercado de games. A ferramenta possibilita explorar diferentes perspectivas dos dados, facilitando a identificação de padrões, tendências e oportunidades, apoiando decisões estratégicas baseadas em dados.
